For Windows 2000. These files contain all the sample code from the book. You can browse these files directly from the CD-ROM, or you can copy them to your hard drive and use them as the basis for your own projects. To find the files on the CD-ROM, open the D:\CODE folder. To copy the files to your hard drive, just run the installation program D:\CODE.EXE. The files will be placed on your hard drive at C:\Program Files\C#VB. You will need to have the .NET framework installed on the machine in order to run the samples.
